## How to Run the App
### Development
Install dependencies:

```bash
npm install
```
Start development server:

```bash
npm start
```
Runs on http://localhost:3000 with hot reloading

### Production Build
Create optimized build:

```bash
npm run build
```
Outputs to /dist folder. Contains index.html and bundle.js
